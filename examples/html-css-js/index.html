MapForge Three.js Export Demo with Point Popup

把 MapForge 右側 Copy Full 的 mapProject 貼到 src/mapProject.js，滑鼠移到點位會顯示資訊。

使用方式

  1. 在 MapForge 右側按 Copy Full。
  2. 打開 src/mapProject.js。
  3. 用你的 export const mapProject = {...} 覆蓋原本內容。
  4. 用本機伺服器開啟。

本機啟動

cd MapForge-Export-HTML-CSS-JS-Popup
python3 -m http.server 5173

然後打開 http://localhost:5173

Popup 欄位

點位支援：

{
  id: "pt-1",
  name: "Camera 01",
  type: "camera",
  x: 800,
  y: 180,
  z: 72,
  color: "#ffcc66",
  description: "入口攝影機",
  status: "online",
  floor: "1F"
}

操作

  • 滑鼠左鍵拖曳：旋轉
  • 滾輪：縮放
  • 右鍵拖曳：平移
  • 滑鼠移到點位球體：顯示 Popup
